# The Swift Programming Language (deutsch)

Dies ist eine Übersetzung der [The Swift Programming Language](https://docs.swift.org/swift-book/documentation/the-swift-programming-language/) Dokumentation ins deutsche.

Sie basiert auf der aktuellen Veröffentlichung 
* Swift 6.2.3, 10.12.2025


Um diese Dokumentation im Browser anzuzeigen, führen Sie zuert folgendes Kommando in der Kommandozeile aus. um die Markdown-Datein mit dem [Swift-DocC](https://github.com/swiftlang/swift-docc) zu bauen:
```shell
# Baut die Dokumentation
xcrun docc convert TSPLGER.docc \
--experimental-enable-custom-templates

# Startet den lokalen Vorschau-Server auf dem Port 8000 
# um die Dokumentation im Browser anzusehen. 
xcrun docc preview TSPLGER.docc
```
Danach auf den Link klicken um die lokale Vorschau der Dokumentation zu sehen.
